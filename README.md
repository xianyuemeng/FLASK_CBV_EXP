本项目是一个基于flask的CBV的接口范例
===
---

* 安装环境: `pip install -r requirement.txt`
* 运行: `python run.py`
* 创建数据的`json格式`范例：
    ```json
    {
        "name": "tianqi",
        "pwd": "123456",
        "email": "tianqi@qq.com",
        "age": "22",
        "phone": "1234567",
        "info": "tianqi",
        "face": "tianqi"
    }
    ```
